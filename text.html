囲み枠

jumbotron

ジャンボトロンはWebサイトのタイトルなどに活用できる見出しです

well

ブロック要素に well というクラスを適用すると、その範囲全体を角丸の背景色で囲って表示できます。コラムのような囲み記事を作成する場合などに活用できるでしょう。

well-lg

ブロックを大きくする場合は well-lg を指定します。wellの指定をお忘れなく(well + well-lg)

well-sm

ブロックを小さくする場合は well-sm を指定します。wellの指定をお忘れなく(well + well-sm)

thumbnail

角丸の枠線で囲む場合は thumbnail を指定します

caption を付けると周囲に余白が付きます

定義リストの表示 (dl, dt, dd)

HTML
Web制作の基本となる言語で、Webページに表示する内容（文字や画像、リンク情報など）をタグを使って記述していきます。
CSS
サイズや色、枠線、余白といった書式を指定する言語です。WebサイトのデザインはこのCSSを使って指定していきます。
JavaScript
閲覧者のマウス操作などに合わせて動きのあるWebページを作成する場合に利用する言語です。Javaと名前が似ているが、異なるプログラミング言語である。オブジェクト指向のスクリプト言語であることを特徴とする。実行環境が主にウェブブラウザに実装され、動的なウェブサイト構築や、リッチインターネットアプリケーションなど高度なユーザインタフェースの開発に用いられる。

画像の書式

親要素の幅に合わせて縮小して表示する場合は img-responsive を指定します。画像が小さい場合は拡大されません！

画像の四隅を角丸に(img-rounded)

画像を楕円形で(img-circle)

画像の周囲を角丸の枠線で囲む(img-thumbnail)

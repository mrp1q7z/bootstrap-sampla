Media object

タイトル１

Media objectを利用するときは全体を div で囲み、このdiv要素に media のクラスを適用します。続いて画像を表示する img 要素を a タグで囲み、この要素に media-left のクラスを適用します。

上記のつづき

画像の右側に表示する文章は media-body のクラスを適用した div で囲んで記述します。このとき文章の見出しとなる要素に media-heading のクラスを適用しておくのが基本です。

画像を角丸で

画像を角丸で表示するために img-rounded のクラスを追加してあります。これは必須ではありません。
